The Events Calendar addon to design and modify single event page template. Event details page template builder.

== Description ==

### 📅 The Events Calendar Single Event Page Templates

Install this plugin along with the famous **[🔗 The Events Calendar](https://wordpress.org/plugins/the-events-calendar/)** plugin (800,000+ Active Installations) and extend single event page design limitations of The Events Calendar. By using **Event Single Page Templates Addon**, You can showcase your event details page in a beautiful design, you can design single event page in a color combination by using advanced color settings of this events calendar addon.

### Plugin features

* Customize single event page design.
* Select custom color and date format for event page.
* Advanced setting to implement template design on all events or a single event.
* Fully responsive template page design.
* Show event countdown timer on event single page.
* Show related events below event page.

== Installation ==

1. Install **Event Page Templates Addon** from the WordPress.org repository or by uploading plugin-zip unzipped folder to the **/wp-content/plugins** directory. You must also install **The Event Calendar** free plugin (by Modern Tribe) if you want to use this addon.

2. Activate the plugin through **Plugins >> Installed Plugin** menu in WordPress

3. You can adjust template color and other settings from **Events >> Events Single Page Templates >> Edit**.

== Frequently Asked Questions ==

= How it works? =

This plugin works as an addon for **The Events Calenar** plugin. First you need to install and activate free version of "The Events Calendar" then install this addon and design beautiful single event page for the events created via The Events Calendar (by modern tribe).

= Can I also design all events list/grid view using this addon? =

No, this addon will only design single event pages, if you want to showcase your all events list/grid in a beautiful design then you can use our other addon for The Events Calendar - [The Events Calendar Shortcode and Templates](https://wordpress.org/plugins/template-events-calendar/).

= Can I select any date format on single event page? =

Yes, from template settings you can select any date format that you want to show on single event page.
