# All-In-One Digital Fundraising Solution For WordPress. #

You can build your fundraising website using Crowdfundly to raise money for any reason, e.g., a charity, a non-profit organization, a school, a political campaign, a religious organization, a reward-based campaign, a creative project, or a sponsored campaign for a content creator.


**Note:** [Crowdfundly](https://crowdfundly.com) is a Software as a Service (SaaS) digital crowdfunding solution. The best fundraising solution for WordPress with Elementor support. To integrate Crowdfundly with your WordPress site, you must choose a package. It's free to get started, and you can upgrade at any time.

== Installation ==

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "Crowdfundly". 
3. Install, then Activate it.
4. Follow the [Documentation](https://crowdfundly.com/docs/)

= Old Way: =
1. Upload `crowdfundly` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the [Documentation](https://crowdfundly.com/docs/)
